# Moodle.2

## О системе
Это платформа для управления учебными курсами на базе GitHub.

## Участники
- **Преподаватели:** Создают курсы и проверяют задания
- **Студенты:** Выполняют задания через Pull Requests

## Доступные курсы
- [Introduction to Python](courses/course-intro-to-python/)
- [Web Development 2023](courses/course-web-dev-2023/)
